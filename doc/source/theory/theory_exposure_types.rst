Exposure Types
==============

Two types of exposures can be distinguished. The first type concerns the exposure between two ships. This exposure is denoted as collision candidates. 
The second type is the exposure between a ship and an static object in an area. This exposure is denoted as danger miles. Both types are treated 
separately and are therefore discussed in separate sections. The outcome of an exposure computation is the exposure rate or exposure frequency. The exposure 
rate is the number of exposures per unit time.
