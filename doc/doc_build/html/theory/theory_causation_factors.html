Causation Factors#

The causation factors are derived from the ratio between the observed exposures and the true accidents in an area. Two exposure types are specified, denoted as ship-ship and ship-static object. Therefore causation factors are established for both two types. A distinction can be made according to ship size and type.

The multiplication of the ship-ship exposures (collison candidates) with the causation factor results in a number of predicted events in the area of interest. Ship-static object exposures (danger miles) with the causation factor provides you the number of ships that drift/ram a static object, if no action is undertaken. Measures can be taken to reduce the number of accidents. These measures are discussed in the theory_measures section.

Ship-Ship Causation Factors#

Ships are assigned to links and cells. This is traffic in an area (see section Traffic). The passages per unit time (\(Q_{ij}\)) and the lateral distibution parameters (\(\mu_{ij}\) and \(\sigma_{ij}\)) are specified per link (\(i\)) and ship type (\(j\)). The cell density (\(D_{ij}\)) represents the travelled distance per unit area and unit time. The traffic output properties are used to determine the exposures, denoted as collision candidates for the ship-ship exposure type. The output of a ship-ship exposure model is a rate (number of exposures per time unit).

(59)#\[N_a^b(s,t) \text{ with } a \in {RR,NN,RN,NR} \text{ and } b \in {OV,CR,HO}\]

Special Case#

Suppose that for each combination of \(a,b,s,t\) the number of actual collisions in the area per time unit is known as \(C_a^b(s,t)\). Then we can determine the causation factor using:

(60)#\[F_a^b(s,t) = \frac{C_a^b(s,t)}{N_a^b(s,t)}\]

A dimension analysis of \(F_a^b(s,t)\) results in:

\[\frac{ \frac{\text{number of true collisions}}{\text{time unit}} }{ \frac{\text{number of collision candidates}}{\text{time unit}} }\]

Which can be simplified to:

\[\frac{\text{number of true collisions}}{\text{number of collision candidates}}\]

General Case#

In practice it is often the case that \(C_a^b(s,t)\) is not known for each single combination of \(a,b,s,t\). Let \(G\) be the set of all possible combinations of \(a,b,s,t\).

Suppose that we have a disjunct partition of \(G\) in \(X\) parts:

(61)#\[ G = \bigcup_{x=1}^X G_x\]

And corresponding collision statistics \(C_x\) for each \(G_x\), assuming that the causation factor for all members of \(G_x\) is equal. We can calculate the corresponding collision candidates.

(62)#\[N_x = \sum_{(a,b,s,t) \in G_x} N_a^b(s,t)\]

And the corresponding causation factors:

(63)#\[F_x = \frac{C_x}{N_x}\]

Example

This example serves to illustrate the underlying concept using a minimal configuration. The objective is conceptual clarity rather than quantitative accuracy. The modeled area comprises three pairs of two links.

  1. Link \(A\) and \(B\), connected via one shared waypoint node (\(\theta=30deg\)).

  2. Link \(C\) and \(D\), connected via one shared waypoint node (\(\theta=100deg\)).

  3. Link \(E\) and \(F\), connected via two shared waypoint nodes (\(\theta=180deg\)).

Three ship types (\(i\), \(j\) and \(k\)) operate in the area. The following traffic is composed with ship types assigned to links:

  • Link \(A\): ship \(i\)

  • Link \(B\): ship \(j\) and \(k\)

  • Link \(C\): ship \(i\)

  • Link \(D\): ship \(k\)

  • Link \(E\): ship \(i\)

  • Link \(F\): ship \(j\)

This yields the following set of exposures and some arbitrarily assigned exposure frequency values and fractions. For clarity, we substitute the \(RR\) notation with the link tags.

  • Link pair \(AB\), computed with the crossing ships model:

    • \(N_{AB}^{OV}(i,j) = 10\)

      • The bow of ship \(i\) collides with ship \(j\): \(\frac{D_{i}}{D_{ij}} = 0.6\)

      • The bow of ship \(j\) collides with ship \(i\): \(\frac{D_{j}}{D_{ij}} = 0.4\)

    • \(N_{AB}^{OV}(i,k) = 2\)

      • The bow of ship \(i\) collides with ship \(k\): \(\frac{D_{i}}{D_{ik}} = 0.2\)

      • The bow of ship \(k\) collides with ship \(i\): \(\frac{D_{k}}{D_{ik}} = 0.8\)

  • Link \(B\), computed with the overtaking ships model:

    • \(N_{B}^{OV}(j,k) = 2\)

      • The bow of ship \(j\) collides with the stern of ship \(k\): \(f_j = 0.0\)

      • The bow of ship \(k\) collides with the stern of ship \(j\): \(f_k = 1.0\)

  • Link pair \(CD\), computed with the crossing ships model:

    • \(N_{CD}^{CR}(i,k) = 6\)

      • The bow of ship \(i\) collides with ship \(k\): \(\frac{D_{i}}{D_{ik}} = 0.7\)

      • The bow of ship \(k\) collides with ship \(i\): \(\frac{D_{k}}{D_{ik}} = 0.3\)

  • Link pair \(EF\), computed with the head-on ships model:

    • \(N_{EF}^{HO}(i,j) = 4\)

      • The bow of ship \(i\) collides with the bow of ship \(j\): \(f_i = 0.5\)

      • The bow of ship \(j\) collides with the bow of ship \(i\): \(f_j = 0.5\)

The following observations can be made from this review:

  • The crossing exposure model can be applied to derive overtaking- and head-on exposure frequencies, depending on the value of \(\theta\).

  • The speed of ship \(k\) on link \(B\) is larger than the speed of ship \(j\). Ship \(k\) is for this pair always the striking ship.

  • The computed exposure frequency by the head-on model is equally distributed over both ships.

We list our exposure frequencies of struck ships as follows:

  • \(N_{A}^{OV}(i) = 0.4 \cdot 10 = 4\)

  • \(N_{B}^{OV}(j) = 0.6 \cdot 10 = 6\)

  • \(N_{A}^{OV}(i) = 0.8 \cdot 2 = 1.6\)

  • \(N_{B}^{OV}(k) = 0.2 \cdot 2 = 0.4\)

  • \(N_{B}^{OV}(j) = 1.0 \cdot 2 = 2\)

  • \(N_{B}^{OV}(k) = 0.0 \cdot 2 = 0\)

  • \(N_{C}^{CR}(i) = 0.3 \cdot 6 = 1.8\)

  • \(N_{D}^{CR}(k) = 0.7 \cdot 6 = 4.2\)

  • \(N_{E}^{HO}(i) = 0.5 \cdot 4 = 2\)

  • \(N_{F}^{HO}(j) = 0.5 \cdot 4 = 2\)

The following set of arbitrarily assigned \(RR\) accident frequency values has been recorded in the area:

  • Two (2) accidents between ship \(i\) and ship \(j\) per time unit.

    • The bow of ship \(i\) collides with the stern of ship \(j\) (\(\theta=30deg\)).

    • The bow of ship \(i\) collides with the bow of ship \(j\) (\(\theta=165deg\)).

  • One (1) accident between ship \(i\) and ship \(k\) per time unit.

    • The bow of ship \(k\) collides with the side of ship \(i\) (\(\theta=90deg\)).

To derive the collision frequencies, the total number of recorded accidents was distributed with the following rules:

  • Bow-bow contact: evenly distributed between the two ship types.

  • Bow-stern contact: fully assigned to the struck ship with damage at the stern.

  • Bow-side contact: fully assigned to the struck ship with damage at the side.

Yielding the following results for our example:

  • \(C_{RR}^{OV}(j) = 1\)

  • \(C_{RR}^{HO}(i) = 0.5\)

  • \(C_{RR}^{HO}(j) = 0.5\)

  • \(C_{RR}^{CR}(i) = 1\)

If we group ship types \(i\), \(j\) and \(k\), we can compute the causation factors as follows:

  • \(F_{RR}^{OV}(i,j,k) = \frac{C_{RR}^{OV}(i,j,k)}{N_{RR}^{OV}(i,j,k)} = \frac{1}{4+6+1.6+0.4+2+0} = \frac{1}{14}\)

  • \(F_{RR}^{CR}(i,j,k) = \frac{C_{RR}^{CR}(i,j,k)}{N_{RR}^{CR}(i,j,k)} = \frac{1}{1.8+4.2} = \frac{1}{6}\)

  • \(F_{RR}^{HO}(i,j,k) = \frac{C_{RR}^{HO}(i,j,k)}{N_{RR}^{HO}(i,j,k)} = \frac{1}{2+2} = \frac{1}{4}\)

Herewith the collision frequencies of struck ships can be computed again. The sum of the computed collisions of all involved ships is \(3\).

  • \(C_{A}^{OV}(i) = F_{RR}^{OV}(i,j,k) \cdot N_{A}^{OV}(i) = \frac{4}{14}\)

  • \(C_{B}^{OV}(j) = F_{RR}^{OV}(i,j,k) \cdot N_{B}^{OV}(j) = \frac{6}{14}\)

  • \(C_{A}^{OV}(i) = F_{RR}^{OV}(i,j,k) \cdot N_{A}^{OV}(j) = \frac{1.6}{14}\)

  • \(C_{B}^{OV}(k) = F_{RR}^{OV}(i,j,k) \cdot N_{B}^{OV}(k) = \frac{0.4}{14}\)

  • \(C_{B}^{OV}(j) = F_{RR}^{OV}(i,j,k) \cdot N_{B}^{OV}(j) = \frac{2}{14}\)

  • \(C_{B}^{OV}(k) = F_{RR}^{OV}(i,j,k) \cdot N_{B}^{OV}(k) = \frac{0}{14}\)

  • \(C_{C}^{CR}(i) = F_{RR}^{CR}(i,j,k) \cdot N_{C}^{CR}(i) = \frac{1.8}{6}\)

  • \(C_{D}^{CR}(k) = F_{RR}^{CR}(i,j,k) \cdot N_{D}^{CR}(k) = \frac{4.2}{6}\)

  • \(C_{E}^{HO}(i) = F_{RR}^{HO}(i,j,k) \cdot N_{E}^{HO}(i) = \frac{2}{4}\)

  • \(C_{F}^{HO}(j) = F_{RR}^{HO}(i,j,k) \cdot N_{F}^{HO}(j) = \frac{2}{4}\)

Ship-Object Causation Factors#

The ship-object causation factor is defined as a probability per unit length. The unit length represents a travelled distance by a ship. The probability indicates the occurence of a drift and/or ram event. It can be approached using two different ways:

  1. Define the engine/steering failure probability per nautical mile from literature.

  2. Same as for ship-ship, compute the ratio between the travelled distance and the observed drift- and ram events.

This can be formulated as follows for the first method:

(65)#\[F_{o} = \frac{P_e}{\text{unit length}}\]

Where \({P_e}\) is the probability that an event occurs. This causation factor (\(F_{o}\)) can also be computed with the ratio of observed drift/ram events and the total travelled distances by all ships in an area for a particular time span (second method).

(66)#\[F_{o} = \frac {\frac{\text{total number of observed events}}{\text{time unit}}} {\frac{\text{total travelled distance}}{\text{time unit}}}\]

Which can be simplified to:

\[\frac {\text{total number of observed events}} {\text{total travelled distance}}\]

Once the causation factor in an area has been identified, the ship-object event rates can be calculated using the exposure rates. The ship-object event rate is defined as follows:

(67)#\[C_{o} = F_{o}LQ\]

Where \(L\) is length of the projected segment from a static object on a link and \(Q\) is the number of passages per time unit for each ship type and size for this link. A dimension analysis of (67) results in:

\[\frac{\text{events}}{\text{length unit}}\cdot{\text{length unit}}\cdot{\frac{\text{passing ships}}{\text{time unit}}}\]

Which can be simplified to:

\[\frac{\text{events}}{\text{time unit}}\]
