General
^^^^^^^

:Objective:
  Verify the computed exposure frequency for two vessels located within the same cell.
:Criteria:
  The computed exposure frequency is evaluated by comparing it to the reference value generated by the test script.

The density values of both vessels within the cell serve as input to the exposure models, assuming a distinct 
heading for each vessel. Multiple heading combinations are generated and checked against the analytical 
solution provided by the cell‑crossing model in the test script.
