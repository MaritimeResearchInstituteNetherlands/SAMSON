Rotating Link in Cell
---------------------

General
^^^^^^^

:Objective:
  Verify the computed exposure frequency between a vessel assigned to a cell and a vessel assigned to a link.

:Criteria:
  The computed exposure frequency is assessed by comparing it with the reference value generated by the test script.

A single vessel type is assigned to a cell with a specified density, while the other vessel type is assigned 
to a link with a specified frequency. A unique heading is defined for the vessel in the cell. The link is 
fully enclosed by the cell.

The link is then rotated in multiple incremental steps around its centroid. For each rotation, the exposure 
rate is computed and stored in a data frame. The resulting exposure values must be consistent with the crossing 
model. The shared area corresponds to the blue rectangle shown in the figure.
